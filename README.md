# CNN - Crypto News Network

A premium, AI-driven crypto news broadcast platform.

## 🌟 Features

- **Live AI News Anchor:** Real-time AI character that reads the latest crypto news updates.
- **Dynamic News Desk:** Interactive studio background with multiple camera angles and data displays.
- **Live Market Data:** Real-time price tickers for top crypto assets (BTC, ETH, SOL) and trending meme coins.
- **Sentiment & Fear/Greed:** Integrated market sentiment analysis and altcoin season trackers.
- **RSS News Engine:** Automated aggregation from top crypto news sources (CoinTelegraph, CoinDesk, Decrypt, etc.).
- **Fully Responsive & Optimized:** Highly performant WebP asset pipeline with a mobile-first interface, ensuring the news studio works perfectly on smartphones.

## 🚀 Built With

* **React 18**
* **Vite**
* **Tailwind CSS**
* **Framer Motion**
* **Web Speech API** (for Text-to-Speech)
* **RSS2JSON API** (for live news feeds)

## 💻 Run Locally

**Prerequisites:** Node.js 18+

1. Clone the repository:
   ```bash
   git clone https://github.com/allmightykityy/CNN.git
   ```

2. Navigate into the directory and install dependencies:
   ```bash
   cd CNN
   npm install
   ```
   
3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open `http://localhost:3000` to launch the live broadcast.
